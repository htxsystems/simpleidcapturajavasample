# SimpleID Capture Java Sample
This is a Java Desktop application that shows how to integrate Java code with SimpleID Capture.

## Running the application ##

Before running the application, you need to have a valid installation of SimpleID Core and SimpleID Capture. You also need to set the API Key on SimpleID.java file. Change the following line, build and run the project.

```java
 private String _apiKey = "YOUR_API_KEY";
```
